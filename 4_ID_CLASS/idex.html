id e class

ID

id funciona como um cpf. Assim como so um numero de cpf, cada elemneto do pode ter um valor de id. O cpf e unico , ou seja ningue, pode ter um cpf igual ao meu. Isso significa que o id tambem e unico, entao nao e possivel repetir o mesmo valor duas vezes.

CLASS

O class funciona de maneira oposta ao ID. Diferente do ID em que voce so pode ter um valor, no CLASS voce pode ter varios valores. Tambem diferente do ID, em que nao se pode repetir um valor, no CLASS voce pode utilizar o mesmo valor para varios elementos.
